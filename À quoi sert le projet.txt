À quoi sert le projet ?
Site web pour l’Association Guinéenne des Auditeurs (AGA). Il permet de :
Gérer les membres (adhésions, validations, profils)
Publier et gérer des formations, événements et actualités
Partager des ressources documentaires
Gérer les inscriptions aux formations et événements
Suivre les paiements et cotisations
Communiquer avec les membres via email
Cahier des charges — Site web AGA

2. Architecture technique
2.1 Stack
Framework : Next.js 15 (App Router)
Langage : TypeScript
Base de données : PostgreSQL
ORM : Prisma
Authentification : NextAuth v5 (JWT)
Styling : Tailwind CSS
UI : Radix UI
Email : Resend
Rate limiting : Upstash Redis
Tests : Jest
